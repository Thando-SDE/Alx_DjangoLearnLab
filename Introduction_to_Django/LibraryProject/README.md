# LibraryProject - Django Introduction

This project demonstrates the setup of a Django development environment as part of the ALX Django Learning Lab.

## Project Structure
- `manage.py`: Django command-line utility
- `LibraryProject/settings.py`: Project configuration
- `LibraryProject/urls.py`: URL routing configuration

## Task 0: Django Development Environment Setup
- Installed Django 5.2.7
- Created Django project using `python -m django startproject LibraryProject`
- Successfully ran development server
